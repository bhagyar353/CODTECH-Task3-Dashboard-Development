### Project Overview

This project focuses on developing an interactive data dashboard using Python in Google Colab. The dashboard provides visual insights from the dataset through different charts and graphs.

### Tools and Technologies Used

* Python
* Pandas
* Plotly
* Google Colab
* GitHub

### Visualizations Created

* **Bar Chart:** Total Bill by Day
* **Pie Chart:** Tips Distribution by Gender
* **Scatter Plot:** Total Bill vs Tip Analysis

### Insights

* The bar chart shows how total bills vary across different days.
* The pie chart illustrates the distribution of tips between male and female customers.
* The scatter plot helps understand the relationship between the total bill amount and tips given.

### Conclusion

The dashboard demonstrates how interactive visualizations can help analyze data effectively and provide meaningful insights for decision-making.
